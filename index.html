前期准备

    • 使用包管理工具homebrew

    • 更新为最新版本

  • 安装git

    • 安装完成Node.js后只需要输入下方命令即可

  • 安装Hexo

    • 创建本地文件夹，并使用cd指令进入其中

    • 本地启动

    • 预览页面，使用本地链接http://localhost:4000/

注册github并创建仓库

使用这个文章里所用即可

注意：注册时这里一定要用username.github.io的形式，我自己在注册的时候这里没有注意导致浪费了很多时间

配置ssh

  • 生成ssh key

  • 登录Github，右上角 头像 -> Settings —> SSH nd GPG keys —> New SSH key

    把本目录下xxx.pub文件内容复制粘贴到key中，填好title并点击 Add SSH key

  • 验证

  • 之后还需要设置username和email,为了记录每次提交的记录

  • 注意，我们在配置之后使用下列命令查看密钥列表

  • 若里面没有你使用的密钥，使用如下命令添加

    即上面图中的id_rsa

部署到github

  • 安装部署插件

  • 打开_config.yml文件，在最底部修改部署配置，注意严格的空格（这里指冒号后面一个空格）

    • 注意，这里的repo使用自己库的地址（最好在github上复制下来）

  • 保存后在终端执行

部署完成后可在浏览器输入 yourname.github.io 这样就能看到自己一个属于自己的博客了。
